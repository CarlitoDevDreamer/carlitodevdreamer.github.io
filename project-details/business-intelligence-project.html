PROJECT DETAILS

Business Intelligence Dashboards and Informative ChatBot

This application manages and processes financial and entity information, providing various endpoints for data retrieval, notifications, and a very user-friendly react dashboard and chatbot interface.

Project Overview

Problem:

In our company there was the need to understand the financial information of our clients and their partners in order to assess risks, ensure payment capacity, and make more informed strategic decisions.

Solution:

We contracted a service/API that provides us with information about national and international companies. We developed an application that communicates with this API and stores the data in our database to avoid unnecessary API hits when we know the data hasn’t changed. We also implemented a system of change alerts to reprocess data only when updates occur, helping us avoid costly API calls. Additionally, we set up scheduled tasks to ensure we always have the most up-to-date information possible. A beta chatbot was also created to allow direct queries about entities, offering a much more practical experience, especially for mobile use.

Key Features

  • Entity Management: Create, update, process, and list financial entities with detailed information.
  • Advanced Search: Search entities and their financial data using multiple filters and criteria.
  • Financial Data Retrieval: Access executive summaries, balance sheets, ratios, incidents, judicial actions, and more via dedicated endpoints.
  • Notification System: Manage and update notifications for entities, including filtering by provider, type, and date range.
  • Custom Dashboard Integration: Securely redirect and display entity data in a modern frontend dashboard. (React JS)
  • Bulk Data Operations: Fetch and process multiple entities or products in a single request.
  • Real-Time Data Processing: Endpoints support real-time updates and asynchronous data handling.
  • API-First Design: All features are accessible via RESTful API endpoints for easy integration.
