PROJECT DETAILS

Event Participant Management Application

A real-time, multi-user event check-in solution built for fast, professional participant management.

Project Overview

Problem:

We had an event happening in three days, and the event managers were relying on an Excel sheet containing all participants and relevant information. However, there was a need for multiple users to monitor event attendance in real time, using a more robust, modern, professional, and user-friendly solution.

Solution:

A custom WordPress plugin developed using PHP, JavaScript, and CSS, following an MVC and OOP approach. It allowed multiple receptionists—working remotely on tablets or laptops—to access and update the same real-time data. This was something difficult to achieve with just an Excel sheet. The solution also brought a touch of professionalism and visual polish, significantly improving the experience compared to the outdated method of managing participant check-ins via Excel. (Excel is pretty powerful, by the way!!)

Key Features

  • Check in Checkout: Allows Statistics in real time
  • Search and Filtering: Find the Participants for Check-in Check-out
  • Managing Multiple Events Allows the management of multiple events with different participants
  • Real Time Assyncronous Updated Data: Each user with a device a connection and access to a browser can update and check data in real time
  • Mobile Responsiveness Allows Portability and anywhere access
  • Manual Create, Update, and Delete: Add, edit, or remove individual participants manually
  • Bulk Insert Participants (via additional plugins): Perform SQL bulk inserts for faster data entry
  • Dark Mode Toggle: Provides a more personalized and comfortable user experience
