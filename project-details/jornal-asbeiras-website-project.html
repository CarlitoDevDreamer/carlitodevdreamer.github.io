PROJECT DETAILS

WordPress - Maintained and plugin developemnt Regional Newspaper Website

This project involves maintaining and developing plugins for a regional newspaper website using WordPress, PHP, MySQL, JS and CSS, enhancing its functionality and user experience.

Project Overview

Problem:

This website needs to be maintained daily and updated with new content, plugins, and features to keep it relevant and functional for its users. One of the most relevant issues that need to be addressed is the need for a way that allows editors to create, update, delete ads, and a way to count clicks and views on each ad, as well as a way to manage the ads more efficiently.

Solution:

A plugin and code that detects visual observer intersection with existing ads for view counts and counting clicks per ad so we can create statistics and reports for the ads, allowing editors to manage ads more efficiently. The plugin also allows for easy creation, updating, and deletion of ads directly from the WordPress dashboard. We also need to always be aware of user feedback and constant monitoring, along with the latest WordPress updates and security patches, to ensure the website remains secure and up-to-date.

Key Features

  • Ads CRUD: Create, Read, Update, Delete ads directly from the WordPress dashboard
  • Statistics: View statistics for each ad, including clicks and views
  • User Feedback: Collect and manage user feedback on ads
  • We have certain reserved spots for ads: This allows for a more dynamic and engaging user experience where each spot shows more than one ad using a really soft discrete transition effect
  • This site includes an Event Participant Manager Plugin: This plugin allows for easy management of event participants, including registration, editing, and deletion of participants.

Project Information

  • Client: Diário as Beiras - Grupo Fapricela
  • Duration: 1-2 months
  • Role: Full-Stack Developer
  • Status: Deployed and operational being maintained
  • Maintenance Date: 2024-2025
